Неверное значение даты при указании года в усеченном варианте
Предварительные шаги:
	1. Открыть страницу "Предложить поездку"
Шаги:
	1. Заполнить поля "Откуда выезжаете" и "Куда едете" валидными значениями 
	2. В поле с указанием даты ввести значение, в котором год обозначен двумя цифрами (например, 15/09/18). 
	3. Поля времени заполнить валидными значениями 
	4. Нажать Продолжить
Ожидаемый результат: 
	1. Переход на следующую страницу с формой "Детали поездки"
	2. Дата в информации о поездке соответствует введенной (в данном случае, 15 сент.)
Фактический результат:
	1. Переход на следующую страницу с формой "Детали поездки"
	2. Дата в информации о поездке на два дня раньше введенной (в данном случае, 13 сент.)
Комментарии:
	При выставлении значения дня на 1 число (например, 01/09/18) при переходе к следующей форме вычитается два дня с учетом месяца (в данном случае, 30 авг.)
	



Ошибка "Выберите дату обратной поездки позже даты выезда." при вводе Даты обратной поездки с усеченным годом
Предварительные шаги:
	1. Открыть страницу "Предложить поездку"
Шаги:
	1. Заполнить поля "Откуда выезжаете" и "Куда едете" валидными значениями 
	2. В поле с указанием Даты отправки ввести валидное значение со значением года в полном варианте(например, 30/09/2018). 
	3. В поле с указанием Даты обратной поездки ввести значение на день позже со значением года в усеченном варианте(например, 31/09/18). 
	4. Поля времени заполнить валидными значениями 
	5. Нажать Продолжить
Ожидаемый результат: 
	1. Переход на следующую страницу с формой "Детали поездки"
	2. Дата в информации о поездке соответствует введенной (в данном случае, Дата отправки: 30 авг., Дата обратной поездки: 31 авг.)
Фактический результат:
	1. Нет перехода на следующую страницу из-за ошибки "Выберите дату обратной поездки позже даты выезда."
	2. Дата отправления остается без изменений: 31/08/2018, значение дня второй даты становится меньше на 1, значение года становится равным 0018: 30/08/0018.
	


	
Дублирующееся сообщение об ошибке под значениями дат поездки
Предварительные шаги:
	1. Открыть страницу "Предложить поездку"
Шаги:
	1. Заполнить поля "Откуда выезжаете" и "Куда едете" валидными значениями 
	2. Поля Дата отправки и Дата обратной поездки оставить пустыми 
	3. В полях значений времени даты отправки установить значение минут, значение часов установить пустым  
	4. В полях значений времени даты обратной поездки установить значение часов, значение минут установить пустым  
	5. Нажать Продолжить
Ожидаемый результат: 
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже" 
	3. Под значением Даты отправки сообщение об ошибке "Введите корректную дату поездки туда." 
	4. Под значением Даты обратной поездки сообщение об ошибке "Введите корректную дату поездки обратно." 
Фактический результат:
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже" 
	3. Под значением Даты отправки два раза выведено сообщение об ошибке "Введите корректную дату поездки туда." 
	4. Под значением Даты обратной поездки два раза выведено сообщение об ошибке "Введите корректную дату поездки обратно." 
	


	
При вводе невалидного значения в поле "Промежуточный пункт" нет ошибки под значением
Предварительные шаги:
	1. Открыть страницу "Предложить поездку"
Шаги:
	1. Заполнить поля "Откуда выезжаете" и "Куда едете" валидными значениями
	2. В поле Промежуточный пункт ввести невалидное значение (например, Эъхрпшй)
	3. В поле с указанием Даты отправки ввести валидные значения 
	4. Поля времени заполнить валидными значениями 
	5. Нажать Продолжить
Ожидаемый результат: 
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже" 
	3. Сообщения об ошибке под значением промежуточного пункта:  "Указанное место не найдено. Проверьте написание."
Фактический результат:
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже"
	3. Сообщение об ошибке под значением промежуточного пункта отсутствует




При вводе невалидных значений в поля "Откуда выезжаете" и "Куда едете", ошибка отображается только под первым полем
Предварительные шаги:
	1. Открыть страницу "Предложить поездку"
Шаги:
	1. Заполнить поля "Откуда выезжаете" и "Куда едете" невалидными значениями (например, Откуда выезжаете: Эъхрпшй, Куда едете: Йшпрхъэ)
	2. В поле с указанием Даты отправки ввести валидные значения 
	3. Поля времени заполнить валидными значениями 
	4. Нажать Продолжить
Ожидаемый результат: 
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже", "Не удалось проложить маршрут. Выберите другие пункты прибытия и отправления." 
	3. Сообщения об ошибке под значениями первого и второго города:  "Указанное место не найдено. Проверьте написание."
Фактический результат:
	1. Нет перехода на следующую страницу
	2. Сообщения об ошибке над формой: "Пожалуйста, исправьте ошибки указанные ниже", "Не удалось проложить маршрут. Выберите другие пункты прибытия и отправления." 
	3. Сообщение об ошибке только под значением первым городом.
Комментарии:
	При исправлении значения первого города (в поле Откуда выезжаете) и подтверждении, появляется ошибка под значением второго города
